# Arbeitsanweisung 02 - SQM

## Auftrag

Führe die lokalen Tests einmal selbstständig aus und halte das Ergebnis fest.

## Was du konkret machen sollst

- Starte die für den Einstieg vorgesehenen Testbefehle lokal.
- Notiere, welche Checks oder Tests du ausgeführt hast.
- Wenn Fehler auftreten, dokumentiere sie und grenze die Ursache so weit wie möglich eigenständig ein.

## Ergebnis für die nächste Sitzung

- Du kannst erklären, welche Tests oder Checks lokal gelaufen sind.
- Du kannst den aktuellen Teststand kurz zusammenfassen.
- Du bringst bei Bedarf eine kurze Fehlerdokumentation mit.
- Du kannst erläutern, was du bei Problemen bereits selbst geprüft hast.
